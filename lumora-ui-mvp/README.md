## 🎯 Objetivo da Atividade

O objetivo desta atividade foi evoluir um projeto de engenharia reversa para uma aplicação autoral, utilizando Inteligência Artificial como apoio no desenvolvimento e Firebase como infraestrutura de armazenamento em nuvem.

A proposta consistia em analisar uma ferramenta existente de geração de interfaces neumórficas e transformá-la em um produto próprio, adicionando identidade visual, novos recursos e melhorias funcionais.

---

## 🧩 Sobre o Projeto

Lumora UI é uma ferramenta web de geração visual para estilos Soft UI/Neumorfismo. A aplicação permite personalizar sombras, intensidade, blur, arredondamento e cores de elementos em tempo real, gerando automaticamente o código CSS correspondente.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, além de integração com Firebase Firestore para armazenamento de dados em nuvem.

## ✨ Diferenciais Implementados

### 1. Modo Claro/Escuro

Foi implementado um sistema de alternância entre Dark Mode e Light Mode, permitindo adaptar a interface de acordo com a preferência visual do usuário.

### 2. Presets Inteligentes

A aplicação possui presets rápidos de estilo como Minimal, Soft e Intenso, facilitando a criação de estilos personalizados sem necessidade de ajustes manuais.

### 3. Download Automático do CSS

O usuário pode baixar automaticamente o CSS gerado em um arquivo `.css`, facilitando a reutilização em outros projetos.

### 4. Histórico em Nuvem com Firebase

Os estilos criados podem ser salvos no Firebase Firestore, permitindo persistência de dados e reutilização dos estilos posteriormente.

---

## ☁️ Integração com Firebase

O projeto utiliza Firebase Firestore para armazenar os estilos criados pelo usuário em banco de dados NoSQL na nuvem.

A integração permitiu transformar uma aplicação puramente visual em uma ferramenta mais robusta, moderna e próxima de aplicações reais utilizadas no mercado.

# Questão A — A Formação do Desenvolvedor na Era da IA

O desenvolvimento assistido por IA muda a forma como o engenheiro de software júnior trabalha e aprende. Hoje, ferramentas como Gemini e ChatGPT conseguem gerar códigos rapidamente, fazendo com que o profissional deixe de focar apenas na escrita de sintaxe e passe a atuar de forma mais estratégica.

Nesse cenário, uma competência essencial é a engenharia de prompt, ou seja, a capacidade de criar instruções claras e detalhadas para orientar a IA corretamente. Além disso, o pensamento crítico também se torna indispensável, já que o desenvolvedor precisa analisar, validar e corrigir os resultados gerados pela IA.

Outra habilidade importante é a adaptabilidade, pois as tecnologias evoluem constantemente e o profissional precisa acompanhar essas mudanças para continuar relevante no mercado.

---

# Questão B — Originalidade vs. Plágio Digital

A engenharia reversa assistida por IA pode ser uma ferramenta muito útil para aprendizado e prototipagem. Porém, ela passa a se tornar plágio digital quando uma interface ou sistema é copiado quase integralmente e utilizado como se fosse uma criação original, principalmente para fins comerciais.

Ferramentas de IA facilitam esse processo, mas isso não significa que o trabalho original de desenvolvedores e empresas deva ser desvalorizado. Por isso, é importante estabelecer limites éticos e incentivar que projetos desenvolvidos com IA apresentem adaptações, personalizações e transparência sobre suas referências.

Uma boa solução seria permitir o uso educacional e criativo dessas ferramentas, mas restringir cópias comerciais diretas que reproduzam totalmente a identidade visual e funcional de produtos originais.
