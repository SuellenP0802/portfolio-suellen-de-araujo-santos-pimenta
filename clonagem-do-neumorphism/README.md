# 🧠 Clonagem de Interface com IA — Neumorphism UI

## 🎯 Objetivo da Atividade

Esta atividade teve como objetivo explorar conceitos de engenharia reversa assistida por Inteligência Artificial, utilizando ferramentas generativas para reconstruir um aplicativo web a partir apenas da observação de sua interface visual e comportamento funcional, sem acesso ao código-fonte original.

A proposta consistia em analisar um webapp existente, compreender sua estrutura visual e lógica de interação, e utilizar IA generativa para auxiliar na recriação do sistema utilizando HTML, CSS e JavaScript.

---

## 🌐 Aplicação de Referência

O webapp escolhido como referência foi o:

🔗 https://neumorphism.io/

O site possui um gerador visual de interfaces no estilo Neumorphism (Soft UI), permitindo personalizar sombras, arredondamento, intensidade e aparência visual dos elementos em tempo real.

## 🔍 Processo de Engenharia Reversa

O desenvolvimento foi realizado através da observação detalhada da interface do sistema original, analisando:

* Estrutura visual
* Organização dos componentes
* Comportamento dos sliders
* Estilo neumórfico
* Sombras e iluminação
* Responsividade
* Atualização dinâmica do CSS

Após a análise inicial, foram utilizadas instruções detalhadas em IA generativa para recriar:

* layout
* estilização
* lógica dos controles
* atualização em tempo real
* geração automática do código CSS

O processo envolveu múltiplos refinamentos até que o resultado visual e funcional se aproximasse do comportamento do sistema original.

---

## 🎨 Funcionalidades Implementadas

A aplicação desenvolvida permite:

* Alterar arredondamento do elemento
* Controlar distância das sombras
* Ajustar blur
* Modificar intensidade das sombras
* Selecionar cor personalizada
* Gerar CSS automaticamente
* Copiar código CSS com um clique

---

## 🧠 Aprendizados Obtidos

Durante a atividade foi possível compreender como ferramentas de IA podem auxiliar no desenvolvimento de software moderno, principalmente em tarefas relacionadas à:

* prototipagem
* desenvolvimento visual
* aceleração de interface
* geração de código

Também foi possível perceber que o papel do desenvolvedor não deixa de existir, mas se transforma em uma função mais estratégica, analítica e orientada à validação e refinamento das soluções geradas por IA.

Além disso, a atividade reforçou conhecimentos relacionados à:

* lógica de programação
* estilização avançada em CSS
* manipulação de DOM
* experiência do usuário
* engenharia de prompt

---

## ⚠️ Desafios Encontrados

Durante o desenvolvimento alguns desafios precisaram ser solucionados, como:

* ajustes de contraste visual
* refinamento das sombras neumórficas
* alinhamento dos componentes
* implementação do seletor de cor
* atualização dinâmica das variáveis CSS

Esses problemas exigiram refinamento contínuo das instruções fornecidas para IA e ajustes manuais no código gerado.

---

## ✅ Conclusão

A atividade demonstrou como a Inteligência Artificial pode atuar como ferramenta de apoio no desenvolvimento de software, acelerando a criação de interfaces e auxiliando na implementação de funcionalidades.

Entretanto, também ficou evidente que o conhecimento técnico do desenvolvedor continua sendo essencial para analisar resultados, validar comportamentos, corrigir erros e transformar o código gerado em uma solução funcional e bem estruturada.
